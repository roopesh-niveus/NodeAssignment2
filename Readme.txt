### Node Assignment 2

STEP 1: Create an express server

STEP 2: Create a GET api that returns query param, path param and return both values with info as json

STEP 3: Create a POST api which acccepts a JSON data. Return an array of data.



## Get API (getServer.js) - requirements 

Query Parameters:
  - name (optional): a string value to be included in the response

Path Parameters:
  - id (required): a numeric identifier

Sample:
  - http://localhost:8080/:123  //without query parameter
  - http://localhost:8080/:123?name=roopz  //with query parameter

## Post API (postServer.js) - requirements 

Use Thunder Client extension to pass json data.
